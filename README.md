# Baby Journey

Pregnancy & Baby Milestones — two-in-one tracker for moms.

## Features

**Pregnancy Mode (40 weeks)**
- Week-by-week guide with fruit size comparisons
- Medical checkup tracker
- Weight tracker with line chart
- Kick counter with timer
- Hospital bag checklist

**Baby Mode (0–24 months)**
- 30 milestones across 4 categories (motor, speech, social, cognitive)
- Growth charts (weight, length, head circumference)
- Development progress tracking

**Core**
- Personalized onboarding
- Dark theme with warm rose/lavender/peach palette
- Haptic feedback
- Data export (JSON via ShareLink)
- Local-only storage, no accounts

## Requirements

- iOS 17.0+
- Xcode 15+

## Build

```bash
open BabyJourney.xcodeproj
```
